Whatsapp API Gateway

Library klien WhatsApp yang terhubung melalui aplikasi browser Web WhatsApp

Untuk dapat menggunakan WhatsApp Gateway :


  1. Buka aplikasi WhatsApp di telepon anda;
  2. Klik menu atau setelan, dan pilih perangkat tertaut;
  3. Arahkan telepon anda ke layar ini untuk memindai kode tersebut;
